# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the countdown arc + jar icon with a stopwatch design: cream background, 75% terracotta progress ring, 7-segment "1d" display, and a small crown button.

**Architecture:** Two Android Vector Drawable XML files control the adaptive icon — a background (solid colour) and a foreground (all visible elements). Both are overwritten in-place; the `ic_launcher.xml` / `ic_launcher_round.xml` manifests need no changes. A canonical SVG is also saved as the design source of truth.

**Tech Stack:** Android Vector Drawable XML, SVG

### Task 3: Replace the adaptive icon foreground

**Files:**
- Modify: `androidApp/src/main/res/drawable/ic_launcher_foreground.xml`

Android Vector Drawable does not support `<polygon>` or `rx` on `<rect>`, so:
- Polygons become `<path android:pathData="M x,y L x,y … Z"/>`
- Rounded rects become quadratic-bezier paths

- [ ] **Step 1: Replace the file with the new vector**
